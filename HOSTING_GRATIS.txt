🚀 CARA TERCEPAT - InfinityFree (10 Menit):

LANGKAH 1: DAFTAR
→ Buka https://www.infinityfree.net
→ Klik "Sign Up"
→ Isi email & password
→ Verifikasi email

LANGKAH 2: BUAT HOSTING
→ Login ke Client Area
→ Klik "Create Account"
→ Pilih subdomain: perpustakaanku.infinityfreeapp.com
→ Tunggu 2-5 menit

LANGKAH 3: UPLOAD FILE
→ Login cPanel
→ File Manager → htdocs
→ Upload perpus.zip
→ Extract
→ Pindahkan isi folder public ke htdocs

LANGKAH 4: BUAT DATABASE
→ cPanel → MySQL Databases
→ Create Database: perpus_app
→ Create User: perpus_user
→ Add User to Database

LANGKAH 5: IMPORT DATABASE
→ cPanel → phpMyAdmin
→ Pilih database
→ Import → database.sql

LANGKAH 6: EDIT .ENV
→ File Manager → perpus → .env
→ Update:
   DB_HOST=sqlxxx.infinityfree.net
   DB_DATABASE=ifxxxxxx_perpus_app
   DB_USERNAME=ifxxxxxx_perpus_user
   DB_PASSWORD=your_password

LANGKAH 7: SELESAI!
→ Buka: https://perpustakaanku.infinityfreeapp.com
→ Login: admin@perpus.com / password
